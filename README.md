# MedClf
This repo explores the performance of mainstream vision models—ResNet, VGG, ConvNeXt, ViT, Swin Transformer, and Next-ViT—in medical image classification.
## Data Source
- MedMNIST(https://github.com/MedMNIST/MedMNIST)
